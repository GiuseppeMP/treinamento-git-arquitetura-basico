Trilha de Treinamentos GIT

Parte I - Arquitetura Git, Comandos Básicos
Aprendendo comandos básicos, como git add, git commit, git init, git log.
Parte II - Comandos Intermediários
Aprendendo comandos Intermediários, como git diff, git difftool, git push, git pull, git fetch, git remote.
Parte III - Comandos Avançados
Aprendendo comandos Avançados, como git merge, git rebase, git bysec, git clone, git show, git stash, git tag.
Parte IV - Git Workflows - Git-flow
Aprendendo a associar git-flow ao fluxo de trabalho do time.
Parte V - Git Workflows - Trunked Based Flow
Aprendendo a associar trunked-based ao fluxo de trabalho do time.
Parte VI - Git Hooks, Plugins
Aprendendo a usar puglins como git-flow, gitup, gitk e hooks como coverage e message regex, commit auto-formatter
Parte VI - GitOps & IaC
Aprendendo o poder do GITOps e IaC
